# SQL-España

Este repositorio incluye ejercicios y prácticas de **SQL básico** relacionados con un conjunto de datos con temática de España. El objetivo de este proyecto es practicar y aplicar consultas SQL para resolver diferentes ejercicios sobre una base de datos.

## Descripción

El proyecto contiene scripts `.sql` que incluyen consultas y definiciones de tablas. Los ejercicios están diseñados para reforzar conceptos fundamentales de SQL, como selección de datos, filtrado, agrupaciones y manipulaciones básicas.

## Contenido del repositorio

- Archivos `.sql` con consultas SQL
- Scripts de creación de tablas y datos de ejemplo
- Ejercicios resueltos para practicar

## Tecnologías utilizadas

- SQL (Structured Query Language)
